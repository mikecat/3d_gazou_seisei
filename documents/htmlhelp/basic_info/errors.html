エラーについて

このソフト独自の関数を呼んだときに起こりうるエラーの解説です。

invalid number of arguments
引数の数が違います。
例えば、2個の引数が必要な関数なのに1個しか指定していない場合に発生します。
invalid arguments
引数の型が違います。
例えば、vectorを渡さないといけない引数にnumberを渡している、 vectorのつもりで要素数の違うリストを渡している、 nilが許可されていない引数にnilを渡している、などの場合に発生します。
invalid parameter(s)
パラメータが無効です。
例えば、正の実数を指定しないといけない引数に負の実数を渡すと発生します。
index out of range
指定されたインデックスが範囲外です。
getCommandParam関数に0以下の数や、 getCommandParamNum関数の戻り値より大きい数を渡すと発生します。
too many objects
オブジェクトが多すぎます。
getMaxObjectNum関数が返す数を超えてオブジェクトを配置しようとすると発生します。
color without any objects
オブジェクトが無いのに色を設定しようとしました。
オブジェクトを配置せずにsetPrevObjectColor関数を呼び出すと発生します。
texture without any objects
オブジェクトが無いのにテクスチャを設定しようとしました。
オブジェクトを配置せずにsetPrevObjectTexture関数を呼び出すと発生します。
texture load error
テクスチャファイルが読み込めませんでした。
setPrevObjectTexture関数で指定したテクスチャが存在しないか壊れています。
テクスチャの相対パスはスクリプトファイルがあるディレクトリではなく、 exeファイルのカレントディレクトリが基準になることに注意してください。
mapping without texture
テクスチャが無いのにマッピングを指定しました。
直前のオブジェクトにテクスチャを指定せずにsetPrevTextureMapping関数を呼び出すと発生します。
